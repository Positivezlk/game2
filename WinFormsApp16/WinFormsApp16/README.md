# CertDesk (WinFormsApp16)

Учебное настольное приложение Windows Forms для учета сертификатов электронной подписи, машиночитаемых доверенностей, сотрудников и USB-токенов.

## Как открыть

Откройте файл решения:

```text
WinFormsApp16/WinFormsApp16.sln
```

в Visual Studio 2022.

## Как запустить

1. Дождитесь восстановления NuGet-пакетов.
2. Нажмите **F5** или кнопку **Start**.
3. При первом запуске автоматически создается SQLite-база `certdesk.db` рядом с exe-файлом.

## Стек

- C#
- .NET 8 Windows Forms
- SQLite через `Microsoft.Data.Sqlite`
- ClosedXML для XLSX-отчетов

## Вкладки приложения

- **Главная** — роль пользователя, сводные показатели и записи, требующие внимания.
- **Сотрудники** — поиск, добавление, изменение, архивирование.
- **Сертификаты** — учет сертификатов, статусы, отзыв и архивирование.
- **МЧД** — учет машиночитаемых доверенностей.
- **Токены** — учет носителей, выдача, возврат и списание.
- **Отчеты** — экспорт CSV и XLSX.
- **Аудит** — журнал действий.

## Роли

На вкладке **Главная** выберите роль:

- **Администратор** — доступны все действия, включая очистку аудита.
- **Специалист** — доступны рабочие действия без очистки аудита.
- **Просмотр** — редактирование, выдача и списание отключены.

## Файлы данных и отчетов

- База данных: `certdesk.db` рядом с exe-файлом.
- Отчеты: папка `Reports` рядом с exe-файлом.
- В репозиторий база и сформированные отчеты не добавляются.
