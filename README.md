# CertDesk

CertDesk — настольная информационная система для контроля сертификатов электронной подписи, машиночитаемых доверенностей и USB-токенов сотрудников образовательной организации.

## Описание

Приложение предназначено для рабочего места ответственного специалиста Западного филиала РАНХиГС. Оно обеспечивает централизованный учет сотрудников, сертификатов ЭП, МЧД, токенов, контроль сроков действия, предупреждения о записях, требующих внимания, аудит действий пользователей, экспорт отчетов и резервное копирование локальной базы данных.

## Стек

- C#;
- .NET 8 Windows Desktop;
- Windows Forms;
- SQLite;
- ADO.NET через `Microsoft.Data.Sqlite`;
- BCrypt.Net-Next для хеширования паролей;
- ClosedXML для XLSX-экспорта.

## Требования

- Windows 10/11;
- Visual Studio 2022 с workload `.NET desktop development`;
- .NET 8 SDK;
- серверная СУБД не требуется.

## Как запустить

1. Откройте `CertDesk.csproj` в Visual Studio.
2. Дождитесь восстановления NuGet-пакетов.
3. Запустите проект кнопкой **Start**.
4. При первом запуске рядом с исполняемым файлом автоматически создается `certdesk.db` и заполняется демонстрационными данными.

## Тестовые пользователи

| Логин | Пароль | Роль |
| --- | --- | --- |
| `admin` | `admin123` | Администратор |
| `spec` | `spec123` | Специалист |
| `view` | `view123` | Просмотр |

Пароли в базе данных хранятся только в виде BCrypt-хешей.

## Структура проекта

- `Program.cs` — старт приложения, инициализация БД и открытие формы входа.
- `Data/` — подключение к SQLite, создание схемы и демонстрационных данных.
- `Models/` — модели предметной области.
- `Services/` — бизнес-логика, роли, валидация, аудит, отчеты, резервное копирование.
- `Forms/` — Windows Forms-интерфейс на русском языке.
- `Reports/` — CSV и XLSX-экспорт.
- `README.md` — инструкция по проекту.

## Где находится база данных

Файл базы данных создается автоматически с именем `certdesk.db` в каталоге запуска приложения (`AppContext.BaseDirectory`). Схема БД описана в `Data/schema.sql`.

## Как создать резервную копию

В главном окне выберите пункт меню **Резервная копия**. Приложение скопирует `certdesk.db` в папку `Backups` с именем вида:

```text
certdesk_backup_yyyyMMdd_HHmmss.db
```

Операция фиксируется в журнале аудита.

## Какие отчеты доступны

В разделе **Отчеты** доступны экспорт в CSV с UTF-8 BOM и XLSX:

1. Реестр сотрудников.
2. Реестр сертификатов.
3. Сертификаты, истекающие в течение 30 дней.
4. Реестр МЧД.
5. МЧД, истекающие в течение 30 дней.
6. Реестр токенов.
7. Журнал операций с токенами.
8. Сводка по подразделениям.
